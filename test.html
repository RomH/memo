    Les images

    Il existe 3 formats d'images sur internet :

    • gif
    • png
    • jpeg

    Le jpeg

    Le jpeg a été conçu pour enregistrer des photos.

    Le png

    Le png a été conçu pour enregistrer des illustrations, des logos ou encore des dessins.
    On peut aussi définir des zones transparentes.

    Le gif

    Le gif a été conçu pour enregistrer des ilustrations dont celles qui sont animées.
    On l'utilise surtout pour cette dernière raison.
